# Code of Conduct

## Our Standard

We want MediFlow to be a professional, respectful, and constructive project space.

Participants are expected to:

- communicate respectfully
- assume good intent while giving clear feedback
- focus discussion on ideas, implementation, and outcomes
- help keep collaboration inclusive and productive

Unacceptable behavior includes:

- harassment or personal attacks
- discriminatory language or conduct
- deliberate disruption of collaboration
- sharing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that do not align with this code of conduct.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project communication channels.
