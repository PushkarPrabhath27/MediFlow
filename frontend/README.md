# MediFlow Frontend

This frontend is a `Next.js` application for the MediFlow platform. It provides the operator-facing experience for equipment visibility, sharing workflow coordination, and dashboard monitoring.

## Stack

- `Next.js`
- `React`
- `TypeScript`
- `Tailwind CSS`
- `Lucide React`

## Available Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Local Development

```bash
npm install
npm run dev
```

The app runs locally at `http://localhost:3000`.

## Notes

- The frontend is designed to work with the backend API exposed by the root `docker-compose.yml`
- Use the root project setup for full-stack local development
